🤖 Ask Your PDF: Intelligent RAG Chatbot
A powerful Retrieval-Augmented Generation (RAG) application that allows you to have natural conversations with your PDF documents. By combining the reasoning capabilities of Google Gemini with FAISS vector search and real-time web access, this tool provides accurate, context-aware answers.

✨ Features
Multi-PDF Analysis: Upload multiple files at once and query across all of them simultaneously.

Dual Search Modes:

PDF Only: Strict retrieval from your uploaded documents (ideal for privacy and precision).

PDF + Web Search: Integrates DuckDuckGo to supplement document data with the latest web information.

Advanced AI Stack: * LLM: gemini-3.1-pro-preview for high-level reasoning.

Embeddings: gemini-embedding-2-preview for state-of-the-art semantic retrieval.

Fast Vector Search: Uses FAISS-CPU for near-instant document indexing and similarity search.

User-Friendly UI: A clean, responsive chat interface built with Gradio.
