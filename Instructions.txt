These are the important instrunctions for running Ask-your-PDF in your local machine 
1. Download all files.
2. Copy Gemini API key from Google AI Studio
3. In folder, open .streamlit/secrets.toml file in notepad
4. Paste your gemini api key to "use_your_gemini_api_key"
5. Save secret.toml file 
6. Refresh folder & run run.bat file
7. Then requirements are downloaded
8. Once finished, you will get message like this 'Running on local URL:  http://127.0.0.1:7654'
9. Copy & paste URL address to your browser
10. Upload PDF and ask your questions
